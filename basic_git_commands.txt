	* git clone https://github.com/aandaleon/compbio_demo.git
		* Pull from your own fork
	* git remote -v
		* See branches
	* git remote add upstream https://github.com/gkthiruvathukal/compbio_demo.git
		* Add upsteam (master maintainer)
	* git add aandaleon.txt
		* Add a file or file changes to the repo
	* git config --global user.email "aandaleon@luc.edu"
	* git config --global user.name "Angela Andaleon"
		* Log in
	* git commit -m "added aandaleon.txt"
		* Write a commit message
	* git push origin master
		* Push the commit to the repository
	* git log
		* Shows all commits over the history of the repo
	  * How do you do a pull request on the command line?
	* git pull upstream master
		* Get latest updates from master maintainer's branch 
	* git config --global --list
		* Find the current user's email and name
	* git diff
		* Shows what has changed
	* git status
		* Shows all changes to the repository.
